build instruction:
1) install NodeJS with NPM
2) run in console/terminal under root user:
    2.1) npm i webpack@^2.2.0 -g
    2.2) npm i webpack-dev-server -g (if you need local server)
3) run in console/terminal in project folder:
    3.1) npm i
    3.2) webpack (to build project)
    3.3) webpack-dev-server (under root user to start local server)
